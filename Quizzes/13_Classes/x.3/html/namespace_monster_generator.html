Monster MonsterGenerator::generate ( )
inline

Randomly generates a Monster.

  • Randomly generate a number between 0 and max_monster_types-1 (defined in MonsterType)
    • Use a Mersenne Twister random number generator (defined in random.h) for that.
  • And use that to also index into static arrays of s_names and s_roars
  • hitPoints are randomly generated from 1-100
