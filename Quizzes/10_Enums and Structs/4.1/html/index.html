Animal Locomotor

Given an animal from the user, display how many legs that animal has.

  1. Define an enum class Animal with the animals: pig, chicken, goat, cat, dog, and duck.
  2. Provide these animals as a list of options for the user to select from. The options must be numbered starting from 1.
  3. Overload the operator>> so that you take in an Animal from the user.
  4. Define a function isAnimal() to check for the validity of the input. Keep asking the user for input until they enter a valid animal.
  5. Define an enum Legs inside a namespace Locomotion with the following constants: apod, biped, quadruped, hexaped, octoped.
  6. Define a function getLocomotion() that returns whether a given animal is a biped, or a quadruped, etc. Use a switch case; keep the default value as apod.
  7. Overload the operator<< to print the animal's name and the number of legs it has. Eg: A chicken is a biped. Print "A chicken has 2 legs."
  8. Ask the user if he wants to check for any other animal (y/n). Check for valid input. If 'y', repeat. If 'n', exit.


  • Functions pertaining to animals must be in animals.cpp
  • Functions pertaining to locomotion must be in locomotion.cpp
  • Functions pertaining to i/o must be in io.cpp
  • Each source code file must have its corresponding header file.
