Bitwise Rotate Left


Chapter O.2
A bitwise rotation is like a bitwise shift, except that any bits shifted off one end are added back to the other end. For example 0b1001u << 1 would be 0b0010u, but a left rotate by 1 would result in 0b0011u instead.

  • Implement a function that does a left rotate on a std::bitset<NUM_BITS>.
  • DO NOT use test() and set() included in the bitset header file.
  • Use bitwise operators.
